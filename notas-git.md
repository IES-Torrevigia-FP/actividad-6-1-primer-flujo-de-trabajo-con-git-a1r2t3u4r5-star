# Notas de Git – Actividad 6.1

- Concepto 1: Directorio de trabajo (working directory) → Aquí es donde editamos los archivos normalmente en nuestro sistema.
- Concepto 2: Zona de staging (staging area / index) → Es la zona intermedia donde seleccionamos qué cambios se incluirán en el próximo commit.  
- Concepto 3: Historial de commits (repository / commit history) → Registro permanente de todos los cambios confirmados en el proyecto.
