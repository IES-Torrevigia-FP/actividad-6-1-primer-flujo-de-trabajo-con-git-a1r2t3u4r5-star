En la rama main
Tu rama está actualizada con 'origin/main'.

Cambios a ser confirmados:
  (usa "git restore --staged <archivo>..." para sacar del área de stage)
	borrados:        notas-git.md

Archivos sin seguimiento:
  (usa "git add <archivo>..." para incluirlo a lo que será confirmado)
	cp2-estado-inicial.txt
	notas-git.md

Un archivo untracked es un archivo que Git detecta en el directorio de trabajo, pero que aún no está siendo seguido ni incluido en el historial del repositorio. Para que Git empiece a controlarlo, es necesario usar el comando git add.
